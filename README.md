*[22,27,16,2,18,6] -> Insertion Sort

İnsert Sort'a göre aşamaları:
  -[2,27,16,22,18,6]  -> n
  -[2,6,16,22,18,27]  -> n-1
  -[2,6,16,22,18,27]  -> n-2
  -[2,6,16,18,22,27]  -> n-3
  -[2,6,16,18,22,27]

*Big-O Gösterimi
  n(n-1) / 2  = n2

  O(n2)


*Time Complexity
  Cevap: Average Case
